## 📝 Getting Started

### Prerequisites

- Java 8 or higher
- Maven
- Spring Boot 2.x or higher

### Running Locally

1. Clone the repository
   ```bash
   git clone https://github.com/yaswanth1815/college_management.git
   cd college_management
   ```

2. Build the project
   ```bash
   mvn clean install
   ```

3. Run the application
   ```bash
   mvn spring-boot:run
   ```

4. Access the application at `http://localhost:8080`
